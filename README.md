# HealthCity
A web app that tracks everything regarding your Heath-Records.

A web app that helps hospitals and governments track health related data of users and carry out analytics and Implemented Data Analysis Of Diseases Spread (Area Based, City Based). Visualized disease outbreaks across regions in the state across itâ€™s regions to help authorities. Generates Health-Card for every user


## Steps to run a web app
1) Clone this project 
2) insert health_city.sql in your phpmyadmin database
3) fire up a server and access this folder in your htdocs folder
